# CampusEatS-Online-Canteen-Food-Ordering-For-MET-Campus
Designed and developed to solve our MET campus’s real-life queue and order-management problem by bringing the entire canteen food-ordering process online. • Built with HTML, CSS, JavaScript, Flask (Python), and MySQL (phpMyAdmin) to offer a smooth, simple, and fast ex- perience for both students and canteen staff.
